# Netlify를 이용한 배포

## 정적, 동적 페이지
#### * 정적 페이지(Static page)
#### : 시간이나 사용자의 요청에 따라 변하지 않는 페이지 
####  -> HTML
#### * 동적 페이지
#### : 시간이나 사용자의 요청에 따라 변하는 페이지 
####  -> HTML + CSS + JS

#### 2. New site from Git
